The dialog editor can be opened via SpyOnHuman/DialogEditor
New characters can be created in the CharcterEditor: SpyOnHuman/CharacterEditor

Use of Dialog Editor:

Create new dialog via Data/New
Add new nodes via right mouse button
Move the view via the middle mouse button
And move nodes or create connection via the left mouse button

The right mouse button opens a context menu on connections, handles and nodes as well

Create a new Node Script:

Right click on a folder - the node should be in
Click Create/Node C#
A window pops up which allows you to create the visuals of the node
When you are ready click create

!!! (This function is highly new and not finished or bug tested)

Place node scripts in the SOH/Scripts/DialogNodesLibrary/ folder
